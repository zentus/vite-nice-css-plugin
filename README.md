# @zentus/vite-nice-css-plugin

## Installation

`npm install @zentus/vite-nice-css-plugin`

## Usage

Add the plugin in your `vite.config.js`

```js
import viteNiceCssPlugin from '@zentus/vite-nice-css-plugin'

export default defineConfig({
  plugins: [
    react(),
    viteNiceCssPlugin()
  ]
})
```
